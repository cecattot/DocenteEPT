Registro Digital - Desenvolvimento de Software

Neste registro digital, narramos as dificuldades enfrentadas durante o experimento de desenvolvimento de software.

Dificuldades Encontradas:

Durante o experimento, os alunos enfrentaram algumas dificuldades, especialmente nas seguintes áreas:

  • Compreensão inicial dos conceitos de HTML e CSS.
  • Formatação CSS para tornar o layout responsivo em dispositivos móveis.
  • Resolução de problemas durante a implementação da aplicação web.

Superando os Desafios:

Apesar das dificuldades, os alunos conseguiram superar os desafios com:

  • Ajuda dos materiais de apoio disponíveis.
  • Orientação e suporte do professor durante o processo.
  • Trabalho em equipe e colaboração mútua entre os membros do grupo.

Este registro digital reflete a jornada de aprendizado e crescimento dos alunos no desenvolvimento de software.
